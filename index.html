What is the difference between var, let, and const?** `var` is function‑scoped, can be redeclared, and hoisted. `let` is block‑scoped, can be updated but not redeclared. `const` is block‑scoped, cannot be updated or redeclared, and must be initialized at declaration. **2. What is the spread operator (...)?** The spread operator expands elements of an iterable (like array or object) into individual items. Example: `const newArr = [...oldArr, 4]` or `const newObj = {...oldObj, key: 'value'}`. **3. What is the difference between map(), filter(), and forEach()?** `map()` returns a new array after transforming each element. `filter()` returns a new array with elements that satisfy a condition. `forEach()` executes a function on each element but returns nothing (used for side effects). **4. What is an arrow function?** Arrow functions are a concise syntax: `(param) => expression`. They don't have their own `this`, `arguments`, or `super`, and cannot be used as constructors. **5. What are template literals?** Template literals are strings enclosed by backticks (`) allowing embedded expressions `${}` and multi‑line strings, making string concatenation cleaner and more readable.
